_请注意，按此处描述重置钱包时，**凡是并非根据助记词生成的账户（地址），之后将不予恢复**。 这些导入的账户，无论是硬件钱包还是使用私钥导入，都不包含在助记词中，需要[手动重新添加](/managing-my-wallet/accounts-and-addresses/how-to-add-missing-accounts-after-restoring-with-secret-recovery-phrase)。 请务必确认您拥有导入账户的私钥/助记词，再继续重置过程。_

重置 MetaMask 钱包的密码时，您将需要**12 个词的助记词**。如果您尚未妥善保存助记词，请务必阅读[如何显示助记词](/privacy-and-security/how-to-reveal-your-secret-recovery-phrase)。

**执行这些步骤之前，请确认您已准备好助记词，否则您将无法重置密码，且或将无法访问钱包。**

<Tabs>

<TabItem value="扩展版" label="扩展版">

1. 如果您的钱包当前已解锁，请将其锁定，点击右上角的**账户**图标，打开下拉菜单，在账户下拉菜单中选择**锁定**。

2. 在解锁视图中，单击**忘记密码：**

5. 重新输入密码，以确认密码书写正确。
6. 单击**恢复**：

<TabItem value="移动版" label="移动版">

1. 如果您的钱包当前未锁定，您需要锁定应用程序才能访问重置密码的按钮。为此，单击**侧栏菜单**，然后选择底部的**锁定**。

2. 进入解锁视图后，单击**重置钱包**。

3. 您将看到以下消息：**“您确定要清空您的钱包吗？”**单击**“我明白，继续”。**

4. 键入**删除**我的钱包。

5. 您确认之后，将返回到设置屏幕，然后点击**使用助记词导入**。

**MetaMask 的支持部门能否为我重置密码？**

MetaMask 不是基于云的解决方案，绝无权限访问您的助记词。助记词驻留在本地设备（客户端）上，**仅您可见**。

如果您收到助记词无效的错误，请查看[此处](/privacy-and-security/how-to-fix-invalid-seed-phrase-error-invalid-secret-recovery-phrase)。

有关 MetaMask 中密码的更多信息，请参见[此处](/privacy-and-security/passwords-and-metamask)。
