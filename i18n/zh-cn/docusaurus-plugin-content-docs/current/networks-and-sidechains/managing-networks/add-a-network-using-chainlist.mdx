---
title: "使用链列表添加网络"
---

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

要将网络添加至 MetaMask，一个简单方法就是访问 Chainlist。

<Tabs>

<TabItem value="扩展版" label="扩展版">

1. 前往 Chainlist 网站：[chainlist.wtf](https://chainlist.wtf/)。

2. 首先，您需要将钱包连接到该网站。选择右上方的“连接钱包”即可。

   {" "}

   <div class="center-container">
     <div class="img-large">
       <img
         src={
           require("../../../../../../img/networks-and-sidechains/managing-networks/add-a-network-using-chainlist/Connect_to_chainlist_1.png")
             .default
         }
         alt="连接到链表"
       />
     </div>
   </div>

   成功后，您的钱包地址将取代“连接钱包”按钮：

   {" "}

   <div class="center-container">
     <div class="img-medium">
       <img
         src={
           require("../../../../../../img/networks-and-sidechains/managing-networks/add-a-network-using-chainlist/Connect_to_chainlist_2.png")
             .default
         }
         alt="连接到链表"
       />
     </div>
   </div>

3. 现在使用搜索栏查找您要连接的网络。找到后，单击“添加链”按钮。

   {" "}

   <div class="center-container">
     <div class="img-small">
       <img
         src={
           require("../../../../../../img/networks-and-sidechains/managing-networks/add-a-network-using-chainlist/Connect_to_chainlist_3.png")
             .default
         }
         alt="Connect to chainlist.png"
       />
     </div>
   </div>

4. MetaMask 将立即操作，显示批准菜单。单击“批准”以继续。

   {" "}

   <div class="center-container">
     <div class="img-small">
       <img
         src={
           require("../../../../../../img/networks-and-sidechains/managing-networks/add-a-network-using-chainlist/Connect_to_chainlist_4.png")
             .default
         }
         alt="Connect to chainlist.png"
       />
     </div>
   </div>

5. 之后会出现一个菜单，询问您是否要立即切换到新网络。根据需要确认或拒绝；无论您做何选择，都将添加新网络。

</TabItem>

<TabItem value="移动版" label="移动版">

1. 使用[MetaMask 应用内浏览器](https://chainlist.wtf/)，前往[链列表网站](/getting-started/how-to-use-the-metamask-mobile-browser)。

   {" "}

   <div class="center-container">
     <div class="img-medium">
       <img
         src={
           require("../../../../../../img/i18n/legacy/MetaMask_open_mobile_browser.gif")
             .default
         }
         alt="MetaMask 打开移动浏览器"
       />
     </div>
   </div>

   在浏览器中，点击顶部菜单栏中的地址，导航至 [https://chainlist.wtf/](https://chainlist.wtf/)，或点击底部工具栏中的放大镜，然后搜索。

2. 在 Chainlist 上，您现在需要连接您的钱包。您当前选择的账户将是自动连接的账户，不过您可以点击右上角的账户图标，然后再次选择“连接钱包”，便可轻松切换。

   {" "}

   <div class="center-container">
     <div class="img-medium">
       <img
         src={
           require("../../../../../../img/i18n/legacy/Connect_to_chainlist_mobile_1.png")
             .default
         }
         alt="连接到链表"
       />
     </div>
   </div>

   钱包连接后，“连接钱包”按钮所在位置将显示钱包地址。

3. 接下来，通过搜索找到您要连接的网络。找到网络后，单击“添加链”：

   {" "}

   <div class="center-container">
     <div class="img-small">
       <img
         src={
           require("../../../../../../img/networks-and-sidechains/managing-networks/add-a-network-using-chainlist/Connect_to_chainlist_3.png")
             .default
         }
         alt="连接到链表"
       />
     </div>
   </div>

4. 现在您只需要批准添加网络。

   然后会有一个后续选项，确认您是否要立即切换到新的网络。这一步是可选的，您可以随时轻松地更改网络。

<div class="center-container">
  <div class="img-small">
    <img
      src={
        require("../../../../../../img/networks-and-sidechains/managing-networks/add-a-network-using-chainlist/Connect_to_chainlist_mobile_2.png")
          .default
      }
      alt="Connect_to_chainlist_mobile_2.png"
    />
  </div>
</div>

</TabItem>

</Tabs>

有关手动输入详细信息来添加自定义网络的详情，请参阅关于此主题的[文章](/networks-and-sidechains/managing-networks/how-to-add-a-custom-network-rpc)。

**请注意，我们并不维护 Chainlist 上的网络信息。添加网络之前，请确保**网络详情准确无误。
