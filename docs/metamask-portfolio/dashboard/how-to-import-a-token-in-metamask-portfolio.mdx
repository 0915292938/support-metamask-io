#### Want to add a token to MetaMask?

This article is specifically about adding tokens to your portfolio in MetaMask Portfolio at [portfolio.metamask.io](https://portfolio.metamask.io/). If you're looking to add tokens to your MetaMask wallet, see [here](/managing-my-tokens/custom-tokens/how-to-display-tokens-in-metamask).

:::

When you add your accounts to MetaMask Portfolio, most tokens will be detected automatically and displayed in your dashboard.

If you hold a token in one of your accounts and it doesn't automatically show up in your dashboard, you can add it manually.

<Tabs>

<TabItem value="Search for a token" label="Search for a token">

Hit the 'More' button, next to the 'Networks' selector:

<div class="center-container">
  <div class="img-large">
    <img
      src={
        require("../../../img/metamask-portfolio/dashboard/how-to-import-a-token-in-metamask-portfolio/MetaMask_Portfolio_import_token_button.png")
          .default
      }
      alt="MetaMask Portfolio import token button"
    />
  </div>
</div>

This will bring up a menu that lets you search for the token:

<div class="center-container">
  <div class="img-medium">
    <img
      src={
        require("../../../img/metamask-portfolio/dashboard/how-to-import-a-token-in-metamask-portfolio/Import_token_MetaMask_portfolio_dapp5.png")
          .default
      }
      alt="Import token MetaMask portfolio.png"
    />
  </div>
</div>

Input its name, click on it, and then follow the prompts to add it to your assets list.

You won't be able to add a token unless you hold it in one of your connected accounts.

</TabItem>

<TabItem value="Custom token" label="Custom token">

Hit the 'More' button, next to the 'Networks' selector:

<div class="center-container">
  <div class="img-large">
    <img

      }
      alt="MetaMask Portfolio import token button"
    />
  </div>
</div>

From here, select the "Custom Token" tab and insert the details of the token you're looking for. If you need to find the details, head to the block explorer of the relevant network and search for the token by its name or symbol.
