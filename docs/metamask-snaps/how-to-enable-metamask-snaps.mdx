MetaMask Snaps is the key to unlocking a whole array of new and exciting features for your MetaMask experience.

To get started with MetaMask Snaps, ensure MetaMask Extension is updated to **at least** **version 11.0**. [Learn more about updating MetaMask](/managing-my-wallet/using-metamask/how-to-update-the-version-of-metamask). Be aware that MetaMask Snaps is currently only available for the MetaMask browser extension.
